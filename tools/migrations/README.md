# tools/migrations — одноразовые скрипты миграций и диагностики

Скрипты в этой папке вызываются вручную через URL для разовых задач:
инициализация БД, применение индексов, миграции данных, диагностика.

В обычном flow приложения они **не подключаются** (никаких `require`/`include`
из основного кода нет), поэтому раньше лежали в корне, замусоривая список
файлов. Перенесены сюда — но логика и URL-доступ остались.

## Доступ через URL

Папка `tools/migrations/` и каждый `.php` в ней доступны напрямую:
```
https://panel.account-factory.site/tools/migrations/apply_indexes_safe.php
https://panel.account-factory.site/tools/migrations/check_login_index.php
```
(При условии что веб-сервер не блокирует `tools/` — `.htaccess` в корне
не запрещает её.)

Все скрипты требуют залогиненного юзера (`requireAuth()` в начале каждого).

## Содержимое

### Инициализация (одноразово при разворачивании на новом инстансе)
- `create_tables.php` — создаёт основную таблицу `accounts` и сопутствующие.
- `setup_favorites.php` — создаёт таблицу `favorites` для избранных.
- `setup_trash.php` — настраивает мягкое удаление (`deleted_at`).
- `auto_setup.php` — автоинициализация при первом запуске.

### Производительность БД
- `apply_indexes_safe.php` — применяет критические индексы (см. `sql/`).
- `check_login_index.php` — диагностика индексов на колонке `login`.
- `create_optimization_flag.php` — ставит флаг `.optimization_applied`.

### Миграции данных (одноразово при обнаружении проблем)
- `fix_2fa_31digit.php` — очистка 31-значных значений в `two_fa`.
- `fix_gmx_email.php` — нормализация GMX email-адресов.
- `fix_trend_x_error.php` — миграция статусов `trend_x_create_fp` → `trend_x_error`.
- `debug_trend_x.php` — диагностика связанных trend_x статусов.

### Прочее
- `_cleanup.php` — удаление legacy CSS-блока из шаблона.

## Когда удалять?

Когда понимаешь что инициализация/миграция применена на всех инстансах
и точно не понадобится. До этого момента — оставляй: они не нагружают
систему (не подключаются автоматически), но решают разовые задачи быстро.

   - Планируется: XMLHttpRequest с отслеживанием прогресса

4. **Batch INSERT** (Фаза 2.2)
   - Пока каждая строка вставляется отдельно

## Тестирование

### Рекомендации по тестированию

1. **Базовый импорт:**
   ```
   1. Скачать шаблон
   2. Заполнить 10 строк (валидные данные)
   3. Загрузить → должно быть "Добавлено: 10"
   ```

2. **Дубликаты (skip):**
   ```
   1. Импортировать файл с 5 строками
   2. Импортировать тот же файл снова с режимом "Пропустить"
   3. Должно быть "Пропущено (дубликаты): 5"
   ```

3. **Дубликаты (update):**
   ```
   1. Импортировать файл с 5 строками (статус = "active")
   2. Изменить статус на "banned" в CSV
   3. Импортировать с режимом "Обновить"
   4. Должно быть "Обновлено: 5"
   5. Проверить в таблице: статус = "banned"
   ```

4. **Валидация:**
   ```
   1. Заполнить файл без поля "status"
   2. Загрузить → должна быть ошибка "Status is required"
   ```

5. **Большой файл:**
   ```
   1. Создать CSV с 1000 строками
   2. Загрузить → должен импортироваться за <10 секунд
   3. Проверить в логах: только ~11 SQL запросов
   ```

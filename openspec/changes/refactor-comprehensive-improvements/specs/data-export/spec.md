# Spec Delta: Data Export

## MODIFIED Requirements

### Requirement: Экспорт данных в CSV
Система SHALL экспортировать данные аккаунтов в CSV формат, используя единообразную логику фильтрации.

**Изменение:** Рефакторинг для использования `AccountsService` вместо дублирования кода.

#### Scenario: Экспорт всех аккаунтов
- **WHEN** пользователь нажимает "Экспорт в CSV" без фильтров
- **THEN** система использует `AccountsService::getAccounts()` для получения данных
- **AND** формирует CSV файл со всеми аккаунтами
- **AND** отдаёт файл для скачивания

#### Scenario: Экспорт с фильтром по статусу
- **WHEN** пользователь применяет фильтр по статусу и экспортирует
- **THEN** система использует `AccountsService::createFilterFromRequest()`
- **AND** применяет фильтр через `FilterBuilder`
- **AND** экспортирует только отфильтрованные записи
- **AND** логика фильтрации идентична основному дашборду

#### Scenario: Экспорт с множественными фильтрами
- **WHEN** пользователь применяет фильтры по email, 2FA и статусу
- **THEN** система использует единую логику `FilterBuilder`
- **AND** все фильтры применяются корректно
- **AND** нет дублирования кода построения фильтров

#### Scenario: Экспорт больших объёмов данных
- **WHEN** пользователь экспортирует >10,000 записей
- **THEN** система использует пагинированную выборку
- **AND** не загружает всё в память сразу
- **AND** формирует CSV потоково

#### Scenario: Единообразие результатов
- **WHEN** пользователь применяет одинаковые фильтры
- **THEN** результат в дашборде и экспорте идентичен
- **AND** используется одна и та же логика `AccountsService`
- **AND** нет расхождений из-за дублирования кода

## REMOVED Requirements

### Requirement: Дублирующая логика построения фильтров
**Причина удаления:** Дублирование ~150 строк кода, которые копируют логику `FilterBuilder`.

**Миграция:** Все фильтры теперь строятся через `AccountsService::createFilterFromRequest()`.

**До:**
```php
// export.php имел свой собственный код построения фильтров
if (isset($_GET['status']) && is_array($_GET['status'])) {
    $statuses = array_filter($_GET['status']);
    // ... 150+ строк дублирующего кода
}
```

**После:**
```php
// export.php использует общий сервис
require_once __DIR__ . '/includes/AccountsService.php';
$service = new AccountsService();
$filter = $service->createFilterFromRequest($_GET);
```

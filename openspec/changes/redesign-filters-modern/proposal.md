# Change: Современный редизайн блока фильтров

## Why

Текущий блок фильтров функционален, но имеет устаревший дизайн и проблемы UX:

1. **Визуальная перегруженность** - слишком много элементов в одной строке
2. **Неясная иерархия** - все фильтры одинаковой важности
3. **Устаревший стиль** - стандартные Bootstrap элементы без кастомизации
4. **Плохая группировка** - нет логической группировки фильтров
5. **Слабая визуальная обратная связь** - неочевидно, какие фильтры активны
6. **Не хватает современных паттернов** - chips, pills, визуальные индикаторы

## What Changes

### Визуальные улучшения

**1. Новая структура блока:**
- Минималистичный заголовок с счётчиком активных фильтров
- Группировка фильтров по категориям (Основные, Дополнительные, Диапазоны)
- Визуальные chips для активных фильтров
- Плавные анимации и transitions
- Современная цветовая схема

**2. Современные UI элементы:**
- **Search input** с иконкой лупы и кнопкой очистки
- **Status selector** с визуальными chips вместо обычных чекбоксов
- **Quick filters** - быстрые кнопки-фильтры (Email, 2FA, Token)
- **Active filters pills** - визуальные таблетки с активными фильтрами
- **Slide-in animation** при открытии/закрытии

**3. Улучшения UX:**
- Счётчик записей для каждого фильтра в реальном времени
- Быстрая очистка отдельного фильтра (крестик на chip)
- Автоприменение фильтров (live search)
- Keyboard shortcuts (Ctrl+F для фокуса на поиск)
- Индикатор "применено N фильтров"

**4. Адаптивность:**
- Оптимизация для мобильных устройств
- Вертикальный layout на маленьких экранах
- Сворачивание в "drawer" на мобильных

## Design Concept

### Современный дизайн в стиле:
- **Google Material Design** - elevation, ripple effects
- **Notion** - минимализм, чистота
- **Linear** - строгость, функциональность
- **Stripe Dashboard** - профессионализм

### Цветовая схема (уже в проекте):
```css
--primary: #2563eb       /* Синий для акцентов */
--gray-50: #f8fafc       /* Светлый фон */
--gray-100: #f1f5f9      /* Фон карточек */
--gray-200: #e2e8f0      /* Границы */
--success: #059669       /* Зелёный для активных */
```

### Ключевые принципы:
- ✅ **Чистота** - больше воздуха, меньше беспорядка
- ✅ **Фокус** - акцент на главном (поиск и статус)
- ✅ **Скорость** - быстрый доступ к частым фильтрам
- ✅ **Обратная связь** - визуальные индикаторы состояния
- ✅ **Плавность** - transitions и animations

## Impact

### Affected specs:
- `ui-filters` (новый) - спецификация современного UI фильтров

### Affected code:
- `templates/dashboard.php` - HTML структура фильтров (строки 1710-1924)
- `assets/css/` - новые стили для современных фильтров
- `assets/js/dashboard.js` - улучшенная логика фильтров

### Breaking changes:
- **НЕТ** - все существующие фильтры продолжают работать
- Только визуальное обновление, логика не меняется

### UX improvements:
- ✅ Чистый, современный дизайн
- ✅ Визуальные chips для активных фильтров
- ✅ Группировка по категориям

- ✅ Адаптивность для мобильных

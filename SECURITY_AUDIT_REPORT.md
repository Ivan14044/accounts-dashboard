Создайте файл `.env.example` с примером конфигурации (без реальных данных):

### 3. Использовать git-secrets

Установить и настроить `git-secrets`:

```bash
# Установка (macOS)
brew install git-secrets

# Установка (Linux)
git clone https://github.com/awslabs/git-secrets.git
cd git-secrets
sudo make install

# Настройка для проекта
cd /path/to/your/project
git secrets --install
git secrets --register-aws

# Проверка перед коммитом
git secrets --scan
```
